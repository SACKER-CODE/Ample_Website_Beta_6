Dev_Era

Dev_Era is an exciting solo and team coding competition designed to test your development skills through two challenging rounds. Participants will engage in activities that promote creativity and technical proficiency.

Join us for an exhilarating experience where you can showcase your skills and compete for amazing prizes!

Incharge: Sisindri
Time: 3 Hours
Place: Seminar Hall
Entry Fee: ₹150
Prizes: Prize Money + SWAGs + Winner Certificate

Event Overview

Round 1

  • Activity: Developing CSS code from given HTML code.
  • Duration: 50 minutes, followed by 15 minutes for evaluation and a 15-minute break.

Round 2

  • Activity: Develop or write code using the UI design displayed on the screen.
  • Duration: Approximately 50 minutes.
