Code Slayer

Code Slayer is a solo coding competition designed to test your programming skills and problem-solving abilities. Participants will engage in two rounds of challenging activities aimed at enhancing their coding proficiency.

Join us for an exciting experience where you can showcase your skills and compete for great prizes!

Time: 3 Hours
Place: Seminar Hall
Entry Fee: ₹50
Prizes: Prize money+ SWAGs + Winner Certificate

Event Overview

Round 1

  • Activity: Matching code with the given output.
  • Duration: 50 minutes, followed by 15 minutes for evaluation and a 15-minute break.

Round 2

  • Activity: Decode or recreate in-built functions in any programming language.
  • Duration: Approximately 50 minutes.
