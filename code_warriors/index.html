Code Warriors

The event aims to assess participants' knowledge and skills in problem designing, problem solving and problem identification across three levels

Event Summary:

  • Level 1:ebugging or solving the problem in the program which was given in basic C programming.
  • Level 2: Write a code to the given programs in whatever the language you are familiar wihin the limit time. Programs are given easy to medium solving range
  • Level 3: Write a program which is given with in the limit time. Program will be given in tough range(Any Programming Language).

How the Event is Conducted

Level 1:Debugging[First Round]

  • Question papers will be evaluated manually by the faculty.
  • Single participant with tasks(Find the errors and rewrite the program).
  • It is a simple Prelims round filtering the participants for another level

Level 2: Written test (2 - programs)

  • Questions will be given by the faculty
  • Write the program and get the input at runtime and pass the test case
  • Faculty will filter the participant for another level.

Level 3: Written Test (1 program , Final Round)

  • Question will be given by the faculty.
  • The program will be given toughest. Who will complete the program first will be the winner.
